# NMapsGeometry

## Overview

NAVER에서 SPM 지원 안 해줘서 내가 쓸려고 만든 SPM 패키지.

### Requirements

- macOS 13.2.1
- Xcode 14.2

### Installation

#### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/NMapsGeometry.git", .upToNextMajor(from: "1.0.1"))
```

### Usage

```swift
import NMapsGeometry
```
